 E-commerce Website

FashionHub is a responsive fashion e-commerce website built with HTML, Tailwind CSS, and JavaScript.  
It allows users to browse products, add items to cart, and manage authentication using localStorage.


 <!-- Features -->

- Add, remove, and update cart items  
- Dynamic cart counter (real-time updates)  
- Cart persists using localStorage  
- User registration and login system  
- Form validation (email, password, no numbers in names)  
- Show/hide password feature  
- Fully responsive design  


 <!-- Tech Stack -->

- HTML  
- Tailwind CSS  
- JavaScript  
- localStorage
